NOC Config Maker - User Guide
==============================

QUICK START:
1. Double-click NOC-ConfigMaker.exe
2. Wait for browser to open automatically (may take 10-20 seconds)
3. Use the web interface in your browser
4. Keep the console window open while using the application

REQUIREMENTS:
- Windows 10/11
- Internet connection (for AI features)

PORTS USED:
- Port 5000: Backend API (internal)
- Port 8000: Web Interface (your browser)

TROUBLESHOOTING:

Port Already in Use:
- Close other applications using ports 5000 or 8000
- Or restart your computer

Backend Not Starting:
- Check the console window for error messages
- Try running as Administrator

Browser Doesn't Open:
- Manually open: http://localhost:8000/NOC-configMaker.html
- Check Windows Firewall settings

Network Access (for coworkers):
- The application can be accessed from other computers on your network
- Share your IP address and port: http://YOUR_IP:8000/NOC-configMaker.html
- May need to allow ports in Windows Firewall

UPDATES:
When a new version is available:
1. Close the current application (close console window)
2. Replace NOC-ConfigMaker.exe with the new version
3. Run the new executable

SUPPORT:
For issues, check the console window for error messages.
Contact your IT team with any problems.
